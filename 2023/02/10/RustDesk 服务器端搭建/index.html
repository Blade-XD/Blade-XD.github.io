RustDesk 服务器端搭建

发表于2023-02-10|更新于2023-09-14
|阅读量:

ubuntu

环境要求 nodejs & npm & pm2

要注意nodejs 和 npm的安装顺序

https://github.com/rustdesk/rustdesk-server/releases

从github下载服务端文件并解压

切换到文件所在根目录

先在根目录给hbbs和hbbr文件授权

通过pm2调用

key文件id_ed25519.pub会生成在根目录

root用户下设置pm2开机自启调用hbbs和hbbr

windows

NodeJs安装

https://nodejs.org/dist/v16.14.2/node-v16.14.2-x86.msi

pm2安装

  • pm2 start D:\soft\rustdesk-server-windows-x64\hbbs.exe – -r 38.64.92.67 -k _
  • pm2 start D:\soft\rustdesk-server-windows-x64\hbbr.exe – -k _
  • pm2 savepm2 start D:\soft\rustdesk-server-windows-x64\hbbs.exe – -r 38.64.92.67 -k _
    pm2 start D:\soft\rustdesk-server-windows-x64\hbbr.exe – -k _
    pm2 save
