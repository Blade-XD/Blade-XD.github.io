安装前准备

从Arch Linux Downloads下载ISO镜像文件，如 archlinux-2023.11.01-x86_64.iso
如果是物理机需要把ISO镜像文件写入到U盘内

安装过程

进入Live环境 选择Arch Linux install medium (注意：本教程是通过UEFI进行启动安装，请注意UEFI和BIOS字眼)

检测网络，校对时间

分区

选择gpt

选中Free space ，下方选中new选项，回车，将分区空间大小改为1G，然后下方选中Type，选择EFI System

选中Free space ，下方选中new选项，回车，将剩余所有空间分配给该分区，然后下方选择write，输入yes，最后选择Quit退出cfdisk
