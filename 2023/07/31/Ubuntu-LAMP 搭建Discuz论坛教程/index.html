Ubuntu-LAMP 搭建Discuz论坛教程

发表于2023-07-31|更新于2023-09-14
|阅读量:

LAMP详细版本信息

linux版本 Ubuntu 22.04.1 LTS
apache2
mysql
php7.4

安装apache2

修改端口号

修改跟目录

在 /etc/apache2/sites-available 中修改 000-default.conf
中的DocumentRoot /var/www/ 修改为想要的目录

修改默认网页

修改/etc/apache2/mods-available/dir.conf中的内容
DirectoryIndex index.html index.cgi index.pl index.php index.xhtml index.htm

重启服务

安装MySQL5

安装php

discuz不支持php8，要求安装php7.4
若已安装php8，要先彻底卸载错误的php版本

删除php的相关包及配置

删除关联

清除dept列表

检查是否卸载干净（无返回就是卸载完成）

安装php7.4

安装php模块

修改php.ini 显示错误信息

默认情况下是display_errors = Off，把Off修改为On，保存关闭文件,然后重启apache。

重启apache2

下载discuz

解压

拷贝到apache2根目录下

登录mysql

为Discuz!创建专用DB及用户

进入网页开始安装
