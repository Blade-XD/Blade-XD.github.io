发表于2023-08-07|更新于2023-09-14
|阅读量:

搭建环境及安装

操作系统：

win10pro x64

相关环境：

node.js：node-v18.12.1-x86
npm:8.19.2
git:2.41.0.windows.3(64bit)

安装hexo

桌面或在任意windows资源管理器内右键点击 “open git bash here”

安装完成可输入hexo -v查看版本

开始搭建hexo博客

在合适地点新建一个文件夹，例如我在桌面新建一个blog文件夹
并在文件夹内右键打开 “open git bash here”

该功能一般用于本地调试
该命令执行成功后可通过浏览器访问 localhost:4000来查看本地blog站点

部署到gitee(码云)

需要注册账号，国内访问较快，但是需要完成备案，部署详情参考〆清峰ㄟ大佬的教程

部署到github

需要注册账号，国内访问github较为麻烦，需要科学或者魔法，但是不需要备案，切博客部署完可直接访问

左上角点击”+”(加号)，然后点击”New repository”;

在Repository name处输入 “你的ID+.github.io”;

例如我的github id是blade-xd,那么我就需要在这个位置输入blade-xd.github.io;

下面项目可见选项(Public或Private)必须选择 “Public”

注意：这两处位置必须为”你的ID+.github.io”和”Public”

项目创建完毕后可以从github获得一条链接：例如:”https://github.com/Blade-XD/Blade-XD.github.io.git“

最后回到博客文件夹，右键打开“open git bash here”

在git上配置全局邮箱和用户名

把”you@example.com“和”Your Name”改为你的邮箱和用户名

将hexo博客部署到远端站点上

然后通过网页登录或者code的形式进行授权
授权成功后可以看到项目仓库出现刚刚上传的文件
通过链接**”你的ID+.github.io”**(也就是你的项目仓库名称)就可以到部署在github的hexo博客
