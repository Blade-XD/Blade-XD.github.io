1. 使用场景需求

作为打印机发送扫描使用，且不同部门使用不同账户去查看扫描的文件
所有打印机账号(只有上传和查看权限)：printer
不同部门账号(只有对应目录的上传、查看和删除权限)：test1、test2、test3
管理员账号(拥有所有目录的所有权限)：ftpadmin

2. 搭建环境

示例系统：ubuntu server 22.04.3 LTS
示例ftp系统：vsftpd_3.0.5-0ubuntu1_amd64

参考教程：https://blog.csdn.net/ywwwwwwv/article/details/129751296

3. 什么是vsftpd

vsftpd 是 “very secure FTP daemon” 的缩写，安全性是它的一个最大的特点。vsftpd 是一个 UNIX 类操作系统上运行的服务器的名字，它可以运行在诸如 Linux、BSD、Solaris、 HP-UNIX 等系统上面，是一个完全免费的、开发源代码的 ftp 服务器软件，支持很多其他的 FTP 服务器所不支持的特征。比如：非常高的安全性需求、带宽限制、良好的可伸缩性、可创建虚拟用户、支持 IPv6、速率高等。
